# Case Study: DHL Supply Chain — 3-Site Warehouse Consolidation

## Client Profile
**Secteur:** 3PL Logistics  
**Scale:** 21,000 m², 120 employees, 3 sites  
**Clients:** Huawei, OCP, Centrale Laitière, Marjane, Cosumar  
**Timeline:** 24 months (design + implementation + stabilisation)

## Baseline
- Ruptures: 22%  
- SLA compliance: 88%  
- Picking error: 2.2%  
- Picking time: 1.5h per 100 items  

## Intervention & Results
### Phase 1: Warehouse Design (4 weeks)
- DEPICT methodology (DHL standard)
- Multi-sectorial flow analysis
- Capacity planning per site

### Phase 2-3: WMS + Process (12 weeks)
- WMS configuration & testing
- Picking process redesign
- Staff training

### Phase 4: Pilot + Rollout (8 weeks)
- Zone 1 pilot (2 weeks parallel run)
- Full rollout all 3 sites (6 weeks)

### Results (6 months post)
```
Ruptures: 22% → 12% (-45%)
SLA compliance: 88% → 96% (+8pts)
Picking error: 2.2% → 0.6% (-73%)
Picking time: 1.5h → 1h (-33%)
On-time delivery: 94% → 99%
```

## Financial Impact
- Operational efficiency: +25%  
- Cost per unit: -18%  
- ROI: 2.8x | Payback: 5 months
