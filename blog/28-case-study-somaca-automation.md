# Case Study: SOMACA — First iFA Automation in Morocco

**Client:** SOMACA (Renault partner)  
**Project:** Integrated Factory Automation (iFA) + Kitting deployment  
**Scope:** Automation design & implementation  

## Baseline
- Manual warehouse operations  
- Ruptures chaîne: ~15%  
- Kitting: Manual, error-prone  

## Intervention
- iFA design & automation setup  
- Kitting process redesign  
- Integration with production line  
- Staff training (change management)  

## Results
- Ruptures chaîne: 15% → 10% (-33%)
- Kitting errors: High → <1%
- Efficiency montage: +25%
- Productivity: On-plan
